Etapa 3 - Backend

Aplicación web conectada a un servidor

Esta etapa utiliza PHP, una API REST y SQLite para guardar los usuarios de forma permanente mediante operaciones CRUD.

Ir al formulario

Servidor PHP

Procesa las solicitudes HTTP enviadas desde el formulario mediante una API REST.

Base de datos SQLite

Los registros quedan almacenados en una base de datos SQL dentro del servidor.

Operaciones CRUD

Permite crear, consultar, actualizar y eliminar usuarios desde la aplicación.

Servicios del sistema

Gestión de usuarios

Registro, consulta, modificación y eliminación de usuarios.

Persistencia de datos

La información permanece guardada aunque se cierre el navegador.

Seguridad básica

Las contraseñas se guardan mediante hash y no se devuelven al frontend.
